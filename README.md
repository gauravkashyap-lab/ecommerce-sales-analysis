# E-commerce Sales Analysis (SQL Project)

## 📌 Objective
Analyze e-commerce sales data to understand customer behavior and revenue trends.

## 📊 Analysis Performed
- Customer-wise spending  
- Monthly sales trends  
- Top selling products  
- Category-wise revenue  
- Repeat customers  

## 🛠 Tools Used
- SQL Server
- Excel

## 📈 Key Insights
- Few customers generate majority of revenue  
- Seasonal trends impact sales  
- Certain categories dominate revenue
